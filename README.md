# Wordwise Game – Project Overview
• 	Core Idea: A word‑guessing challenge where players try to figure out hidden words, similar to hangman or word puzzles.

## Gameplay:
• 	The player is presented with a word or phrase to solve.

• 	They guess letters or complete words within a limited number of tries.

• 	Correct guesses reveal letters, while wrong guesses reduce chances.

## Tech Stack:
• 	TypeScript for structured, reliable code.

• 	Vite as the fast build tool and dev server.

• 	Tailwind CSS for sleek, responsive styling.

• 	Vitest for testing game logic.

## Purpose:
Fun, educational, and a way to sharpen vocabulary and spelling skills while enjoying a puzzle‑like challenge.
